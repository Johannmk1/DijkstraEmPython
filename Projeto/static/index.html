Visualizador de Grafos & Algoritmo de Dijkstra

Este projeto foi desenvolvido como trabalho da faculdade para mostrar grafos, conceitos de teoria dos grafos e o cálculo do menor caminho usando o algoritmo de Dijkstra. Experimente interagir com o grafo!

Sobre o Projeto

O site permite visualizar grafos com vértices e arestas e calcular o caminho mais curto entre dois pontos usando seu próprio algoritmo de Dijkstra implementado em Python. O grafo é carregado de um banco de dados ou pode ser pré-definido em memória.

O visual é feito com a biblioteca Cytoscape.js, proporcionando uma experiência interativa e intuitiva para explorar o grafo.

Conceitos de Grafos

  • Vértices (nós): representam pontos ou entidades do grafo.
  • Arestas: representam conexões entre vértices, podendo ter pesos (distância, custo, tempo).
  • Grafo não-direcionado: as conexões não possuem direção.
  • Algoritmo de Dijkstra: encontra o caminho mais curto entre um vértice inicial e final.

Como Usar

  1. Selecione o vértice de início e fim.
  2. Clique em Calcular.
  3. O caminho mais curto será exibido abaixo e destacado no grafo.
  4. Experimente diferentes combinações de vértices e explore o grafo interativamente.
